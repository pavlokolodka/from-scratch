# EventEmitter

Basic implementation of Node.js `EventEmitter` object.

## Run tests

### First time

```
git clone https://github.com/pavlokolodka/from-scratch
```

```
npm i
```

### Run tests

```
cd node/event-emitter
npm run test
```
