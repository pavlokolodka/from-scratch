# from-scratch

A collection of different projects, built from scratch for learning purposes.

## Projects

### JavaScript

| Project | Description |
|---------|-------------|
| [Promise](./javascript/promise) | Promises/A+ compliant implementation of the `Promise` object |
| [JSON](./javascript/json) | Native implementation of JSON serialization (`JSON.stringify`) and deserialization (`JSON.parse`) |

### Node.js

| Project | Description |
|---------|-------------|
| [EventEmitter](./node/event-emitter) | Basic implementation of Node.js `EventEmitter` |


### Custom language

| Project | Description |
|---------|-------------|
| [Interpreter](./interpreter) | A custom programming language interpreter implemented in TypeScript |

## Getting started

```bash
git clone https://github.com/pavlokolodka/from-scratch
npm install
```

Then navigate to any sub-project and run its tests:

```bash
cd javascript/promise
npm run test
```
